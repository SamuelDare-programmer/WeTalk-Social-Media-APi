ğŸ› ï¸ Technical Architecture (Under the Hood)
WeTalk leverages a cutting-edge full-stack architecture optimized for performance, scalability, and developer experience.

Frontend (Client-Side)
Framework: React 19 with Vite for lightning-fast development and build performance.
Styling: Tailwind CSS for a completely custom, responsive design system.
Animations: Framer Motion for fluid micro-interactions and page transitions.
State Management: TanStack Query (React Query) for efficient server state synchronization and caching.
Icons: Lucide React for a clean, consistent iconography set.
Routing: React Router DOM v7 for seamless client-side navigation.
Backend (Server-Side)
API Framework: FastAPI (Python) offering high performance and automatic OpenAPI documentation.
Database: MongoDB (NoSQL) accessed via Beanie ODM for flexible data modeling.
Task Queue: Celery with Redis for handling background jobs (e.g., video processing, cleanup tasks).
Media Storage: Cloudinary for optimized image and video hosting/streaming.
Real-time Communication: WebSocket integration for instant messaging feature support.
Deployment & Infrastructure
Hybrid Cloud Strategy:
Vercel for the frontend application and API producer.
Koyeb for heavy background workers and beat schedulers.
